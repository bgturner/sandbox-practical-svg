SVGs as Data URI

Note that we need to url encode the string, ie -- the various brakets need to be converted. So for example we can turn the following:

<svg xmlns="http://www.w3.org/2000/svg" width="150" height="150" viewBox="0 0 10 150"><rect width="25" height="100" rx="5" ry="5" transform="rotate(45)"></rect></svg>

into:

 %3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20width%3D%22150%22%20height%3D%22150%22%20viewBox%3D%220%200%2010%20150%22%3E%3Crect%20width%3D%2225%22%20height%3D%22100%22%20rx%3D%225%22%20ry%3D%225%22%20transform%3D%22rotate(45)%22%3E%3C%2Frect%3E%3C%2Fsvg%3E

and place it into a CSS background style:


#svg-as-data-uri--rectangle {
background: url("data:image/svg+xml,%3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20width%3D%22150%22%20height%3D%22150%22%20viewBox%3D%220%200%2010%20150%22%3E%3Crect%20width%3D%2225%22%20height%3D%22100%22%20rx%3D%225%22%20ry%3D%225%22%20transform%3D%22rotate(45)%22%3E%3C%2Frect%3E%3C%2Fsvg%3E");
width: 200px;
height: 200px;
}
